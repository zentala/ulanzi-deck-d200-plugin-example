# Ulanzi Demo Plugin

Reference implementation of an Ulanzi D200 plugin using the official SDK.
Demonstrates canvas rendering, settings persistence, encoder support, and
lifecycle event handling through three independent actions.

## Actions

| Action | UUID | Description |
|--------|------|-------------|
| Clock | `com.ulanzi.demo.clock` | Live digital clock, updates every second |
| Counter | `com.ulanzi.demo.counter` | Tap to count; hold to reset; dial to adjust |
| CPU Status | `com.ulanzi.demo.status` | CPU usage bar with configurable alert threshold |

## Requirements

- **UlanziStudio** must be running (acts as the WebSocket bridge to the device).
- Node.js / Chromium environment provided by UlanziStudio.
- The two git submodules below must be initialised.

## Installation

1. Add the git submodules:

```bash
git submodule add https://github.com/UlanziTechnology/plugin-common-node \
  ulanzi-demos/plugins/demo/com.ulanzi.demo.ulanziPlugin/libs/plugin-common-node

git submodule add https://github.com/UlanziTechnology/plugin-common-html \
  ulanzi-demos/plugins/demo/com.ulanzi.demo.ulanziPlugin/libs/plugin-common-html

git submodule update --init --recursive
```

2. Create a symlink so UlanziStudio discovers the plugin:

**Windows (run as Administrator):**
```cmd
mklink /D "%APPDATA%\ulanzi\plugins\com.ulanzi.demo.ulanziPlugin" ^
  "C:\path\to\repo\ulanzi-demos\plugins\demo\com.ulanzi.demo.ulanziPlugin"
```

**macOS / Linux:**
```bash
ln -s "$(pwd)/ulanzi-demos/plugins/demo/com.ulanzi.demo.ulanziPlugin" \
  "$HOME/Library/Application Support/ulanzi/plugins/com.ulanzi.demo.ulanziPlugin"
```

3. Restart UlanziStudio. The "Ulanzi Demo" plugin will appear in the action list.

## Assets

PNG icon files (72x72 px) must be placed in `assets/`:

- `plugin-icon.png`
- `clock-action.png`
- `counter-action.png`
- `status-action.png`

These are binary files and are not generated by this repository.

## Project structure

```
com.ulanzi.demo.ulanziPlugin/
├── manifest.json              Plugin descriptor
├── en.json                    Localisation strings
├── assets/                    PNG icons (add manually)
├── libs/
│   ├── plugin-common-node/    SDK bridge – add as submodule
│   └── plugin-common-html/    PI bridge  – add as submodule
├── plugin/
│   ├── app.html               Plugin entry point
│   ├── app.js                 Event dispatcher
│   └── actions/
│       ├── BaseAction.js      Abstract base class
│       ├── ClockAction.js
│       ├── CounterAction.js
│       └── StatusAction.js
└── property-inspector/
    ├── clock/inspector.html
    ├── counter/inspector.html
    └── status/inspector.html
```
